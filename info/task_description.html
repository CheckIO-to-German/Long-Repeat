You have to find the length of the longest substring which consists of a single letter.

An example:
The line "aaabbcaaaa" contains four substrings consisting of only one letter: "aaa", "bb", "c" and "aaaa".
The last of these substrings – "aaaa" – is the longest one, so – since you have to return the length of the longest substring – the correct answer is 4.

Input: A single line as a string.

Output: The length of the longest substring as in int.
